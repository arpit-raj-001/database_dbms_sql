## 📖 Definition

> [!NOTE]
> An **attribute** is a property or characteristic that describes an entity or a relationship.

### Characteristics of Attributes
Every attribute has:
- A name
- A value
- A domain *(allowed values)*
- A data type *(during implementation)*

---

## 🗂️ Attribute Classifications

### 1️⃣ Simple Attribute
A Simple Attribute (Atomic Attribute) cannot be divided into smaller meaningful parts.

### 2️⃣ Composite Attribute
A Composite Attribute can be divided into smaller meaningful attributes.

### 3️⃣ Single-valued Attribute
An attribute that has only one value for each entity.

### 4️⃣ Multivalued Attribute
An attribute that can have multiple values for one entity.

### 5️⃣ Derived Attribute
A Derived Attribute is calculated from another attribute. It is usually not stored because it can be computed when needed.

### 6️⃣ Key Attribute
A Key Attribute uniquely identifies every entity.

### 7️⃣ Null Attribute
A Null Attribute is an attribute whose value is unknown, unavailable, or not applicable.
